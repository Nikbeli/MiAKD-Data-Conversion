Функции true() и false() обычно используются для проверки некоторых условий в процессе программирования.

Преобразование 1

Исходный XML
<source>

<number>0</number>
<number>1</number>

</source>

Результат
<P>true not false</P>

<P>true not false</P>

Представление HTML

true not false

true not false
Преобразование XSLT
<xsl:stylesheet version = '1.0'
     xmlns:xsl='http://www.w3.org/1999/XSL/Transform'>

<xsl:template match="number">
     <P>
          <xsl:if test="true()">
               <xsl:text>true </xsl:text>
          </xsl:if>
          <xsl:if test="not(false())">
               <xsl:text>not false</xsl:text>
          </xsl:if>
     </P>
</xsl:template>


</xsl:stylesheet>
